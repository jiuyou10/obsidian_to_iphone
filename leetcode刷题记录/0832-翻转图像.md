# 832. 翻转图像

🔗 [题目链接](https://leetcode.cn/problems/flipping-an-image/)   📊 难度分 **1243**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:行内翻转+反转合并

## 思路
水平翻转和反转可以合并为一步：对每一行，先逆序，再将每个元素取反（1变0，0变1）。由于元素只有0和1，取反操作等价于1-x。利用列表推导式直接生成新矩阵即可。

## 关键点
- 水平翻转和反转可以同时进行，无需分两步
- 取反操作使用1-x代替if-else判断

## 复杂度
- 时间:O(n^2)
- 空间:O(1)（不计算输出空间）

## 我的代码

```python
class Solution:
    def flipAndInvertImage(self, image: List[List[int]]) -> List[List[int]]:

        return [[1-x for x in reversed(row)] for row in image]
```
