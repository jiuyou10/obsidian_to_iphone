# 713. 乘积小于 K 的子数组

🔗 [题目链接](https://leetcode.cn/problems/subarray-product-less-than-k/)   ⏱️ 耗时 **19 分钟**

> 💡 **关键词**:滑动窗口 + 乘法收缩

## 思路
因为数组元素都是正整数，乘积随窗口扩大单调递增，所以可以用滑动窗口维护一个乘积小于 k 的连续窗口。每次右指针移动后，通过左指针收缩窗口直到乘积小于 k，此时以右指针结尾的满足条件的子数组个数就是窗口长度（即右指针-左指针+1），累加即可。

## 关键点
- 当 k <= 1 时，任何正整数乘积都不可能小于 k，直接返回 0
- 每次右指针移动后，用 while 循环收缩左指针，保证窗口乘积始终小于 k
- 累加的是以当前右指针为结尾的满足条件的子数组个数，即窗口长度

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def numSubarrayProductLessThanK(self, nums: List[int], k: int) -> int:
        if k<=1: #因为这里严格要求乘积<k,且nums都是正整数，所以如果是1也不可以
            return 0
        s=1
        n=len(nums)
        ans=0
        l=0
        #对于r，找到其第一个满足条件的l，这样l-r之间都可以作为左端点
        for i in range(n):
            s*=nums[i]
            while s>=k:
                #r找不到满足条件的l,l右移，缩小窗口，减小乘积
                s/=nums[l]
                l+=1
            ans+=i-l+1
            #如果出现nums[i]>k的情况，那while循环退出时，l=i+1,这里i-l+1=0
        
        return ans
```
