# 2904. 最短且字典序最小的美丽子字符串

🔗 [题目链接](https://leetcode.cn/problems/shortest-and-lexicographically-smallest-beautiful-string/)   📊 难度分 **1483**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+条件收缩

## 思路
循环条件要看清楚，这道题不熟练
用滑动窗口维护恰好包含k个1的子串，当窗口内1的个数超过k或等于k但左端是0时，收缩左边界，从而保证窗口始终是“恰好k个1且左端为1”的最短候选。遍历所有右端点，记录长度最小且字典序最小的子串。

## 关键点
- 滑动窗口内1的个数等于k时，通过while循环将左端多余的0去掉，确保窗口左端是1，从而得到以当前右端点为结尾的最短美丽子串
- 比较候选子串时，先比长度，长度相同再比字典序

## 复杂度
- 时间:O(n^2)（最坏情况下每次截取子串O(n)）
- 空间:O(n)（存储答案子串）

## 我的代码

```python
class Solution:
    def shortestBeautifulSubstring(self, s: str, k: int) -> str:
        
        n=len(s)
        cnt=l=0
        if s.count('1')<k:
            return ''
        ans=s
        for i in range(n):
            cnt+=s[i]=='1'
            while cnt>k or (cnt==k and s[l]=='0'):
                cnt-=s[l]=='1'
                l+=1
            if cnt==k:
                
                t=s[l:i+1]
                print(l,i,t,ans)
                if len(t)<len(ans) or (len(t)==len(ans) and t<ans):
                    print(ans)
                    ans=t
        return ans
```
