# 1248. 统计「优美子数组」

🔗 [题目链接](https://leetcode.cn/problems/count-number-of-nice-subarrays/)   📊 难度分 **1624**   ⏱️ 耗时 **6 分钟**

> 💡 **关键词**:滑动窗口+恰好转至少

## 思路
将“恰好有k个奇数”转化为“至少有k个奇数”减去“至少有k+1个奇数”。使用滑动窗口维护窗口内奇数个数，当奇数个数达到k时，移动左指针使奇数个数刚好小于k，此时左指针的位置就是所有以当前右指针结尾且奇数个数≥k的子数组个数。

## 关键点
- 利用“恰好”与“至少”的转换简化问题
- 滑动窗口统计“至少”时，左指针移动条件为奇数个数≥k

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def numberOfSubarrays(self, nums: List[int], k: int) -> int:
        
        n=len(nums)
        
        #恰好问题可以转为>=->的问题
        def solve(k):
            ans=l=odds=0
            for r in range(n):
                odds+=nums[r]%2
                while l<=r and odds>=k:
                    odds-=nums[l]%2
                    l+=1
                ans+=l
            return ans
        
        return solve(k)-solve(k+1)
```
