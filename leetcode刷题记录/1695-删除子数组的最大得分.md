# 1695. 删除子数组的最大得分

🔗 [题目链接](https://leetcode.cn/problems/maximum-erasure-value/)   📊 难度分 **1529**   ⏱️ 耗时 **8 分钟**

> 💡 **关键词**:滑动窗口+哈希表去重

## 思路
题目要求删除一个元素各不相同的连续子数组，求其最大和。由于子数组必须连续且元素不重复，可以用滑动窗口维护一个无重复元素的窗口，并记录窗口内元素的和。当遇到重复元素时，移动左指针缩小窗口直到重复消除，同时更新窗口和，每次窗口变化后更新最大得分。

## 关键点
- 使用哈希表记录窗口内每个元素的出现次数，判断重复
- 右指针遍历时累加元素和，左指针移动时减去移出元素的值
- 窗口内元素必须全部不同，遇到重复时通过while循环收缩左边界

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
class Solution:
    def maximumUniqueSubarray(self, nums: List[int]) -> int:
        
        l=0
        n=len(nums)
        ans=total=0
        cnt=defaultdict(int)
        for i in range(n):
            cur=nums[i]
            cnt[cur]+=1
            total+=cur
            while cnt[cur]==2:
                cnt[nums[l]]-=1
                total-=nums[l]
                l+=1
            ans=max(ans,total)
        
        return ans
```

## 进阶：如果数组包含负数

> 原题 `1 <= nums[i] <= 10^4` 全为正数。若引入负数，算法需要调整。

### 为什么原解法不适用了

负数会拖累窗口总和，即使没有重复元素，也要考虑丢弃负贡献前缀。

反例：`nums = [5, -8, 6]`

原算法一路扩张得到 `window=[5,-8,6]`，`total=3`，`ans=5`（来自 `[5]`）。但最优实际是 **6**（子数组 `[6]` 单独更大）。

### 思路

对每个右端点 `r`，合法左端点范围是 `[L, r]`（`L` 由去重决定）。要在这个范围内找到最大子数组和：

```
sum(l..r) = prefix[r+1] - prefix[l]
                 ↑固定        ↑找最小值
```

问题转化为：**在滑动窗口 `[L, r]` 内找最小前缀和**，用单调队列维护。

### 代码

```python
from collections import deque
from typing import List

class Solution:
    def maximumUniqueSubarrayWithNegatives(self, nums: List[int]) -> int:
        n = len(nums)
        # 标准前缀和（含前导 0）
        prefix = [0] * (n + 1)
        for i in range(n):
            prefix[i + 1] = prefix[i] + nums[i]

        dq = deque()
        L = 0                     # 去重左边界（能取到的最左位置）
        last = {}                 # 每个元素最近出现的位置
        ans = float('-inf')

        for r in range(n):
            val = nums[r]
            # 更新左边界（去重）
            if val in last:
                L = max(L, last[val] + 1)
            last[val] = r

            # 单调队列：前缀和递增，新来的更小就踢队尾
            while dq and prefix[dq[-1]] >= prefix[r]:
                dq.pop()
            dq.append(r)

            # 移除过期索引（< L 的不可再用）
            while dq and dq[0] < L:
                dq.popleft()

            # 以 r 结尾的最大合法子数组和
            cur = prefix[r + 1] - prefix[dq[0]]
            ans = max(ans, cur)

        return ans
```

### 复杂度
- 时间: O(n)
- 空间: O(n)
