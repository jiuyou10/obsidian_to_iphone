# 2537. 统计好子数组的数目

🔗 [题目链接](https://leetcode.cn/problems/count-the-number-of-good-subarrays/)   📊 难度分 **1892**   ⏱️ 耗时 **16 分钟**

> 💡 **关键词**:滑动窗口+哈希表计数

## 思路
之前自己写的时候多用了个地点，直接变量表示即可，自己做出来的
题目要求统计所有包含至少k对相同元素的子数组。使用滑动窗口维护一个满足条件的窗口，当窗口内相同对数达到k时，固定右端点，左端点可以向左移动的所有位置都对应一个有效子数组。通过哈希表记录元素出现次数，动态更新相同对数。

## 关键点
- 滑动窗口右移时，新增元素与之前出现的相同元素构成新对，更新相同对数
- 当相同对数≥k时，移动左指针缩小窗口，同时更新计数和对数
- 每个右端点对应的有效子数组数量等于左指针的位置（因为左指针左侧的所有位置作为起点都满足条件）

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
class Solution:
    def countGood(self, nums: List[int], k: int) -> int:
        

        n=len(nums)
        ans=l=0
        cnt=defaultdict(int)
        same=0#表示相同对数
        for r in range(n):
            same+=cnt[nums[r]]
            cnt[nums[r]]+=1
            
            while same>=k:
                cnt[nums[l]]-=1
                same-=cnt[nums[l]]
                l+=1
            #从0-l-1
            ans+=l
        return ans
```
