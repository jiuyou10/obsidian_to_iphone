# 1358. 包含所有三种字符的子字符串数目

🔗 [题目链接](https://leetcode.cn/problems/number-of-substrings-containing-all-three-characters/)   📊 难度分 **1646**   ⏱️ 耗时 **7 分钟**

> 💡 **关键词**:滑动窗口+计数统计+l左边计数

## 思路
维护一个滑动窗口，右指针不断右移扩展窗口，当窗口内包含a、b、c三种字符时，左指针右移缩小窗口直到不满足条件。此时，对于当前右指针位置，所有以0到l-1为左端点的子串都满足条件，累加l到答案中。

## 关键点
- 使用哈希表统计窗口内各字符出现次数，当三种字符都至少出现一次时，窗口满足条件
- 每次右指针移动后，通过while循环收缩左边界，记录满足条件的左端点数量

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def numberOfSubstrings(self, s: str) -> int:
        
        n=len(s)
        cnt=defaultdict(int)
        l=ans=0
        for r in range(n):
            #对应每一个r，找其满足条件的最后一个l,那么0-l都可以作为左端点
            #所以这里还是l右移缩小范围
            cnt[s[r]]+=1

            while len(cnt)==3 and min(cnt.values())>=1:
                cnt[s[l]]-=1
                l+=1
            #跳出循环时不满足条件，但是前一次满足条件，也就是l-1
            #从0到l-1都可以作为左边界
            ans+=l
        return ans
```
