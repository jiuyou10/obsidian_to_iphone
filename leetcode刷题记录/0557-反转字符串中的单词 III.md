# 557. 反转字符串中的单词 III

🔗 [题目链接](https://leetcode.cn/problems/reverse-words-in-a-string-iii/)   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:遍历分割+反转拼接

## 思路
记住，字符串也可以切片
因为题目要求保留单词顺序只反转每个单词内部字符，所以可以遍历字符串，遇到空格或结尾时，将之前累积的单词反转并存入结果列表。最后用空格连接所有反转后的单词即可。

## 关键点
- 用临时变量累积当前单词的字符
- 在遇到空格或字符串末尾时触发反转和重置

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
class Solution:
    def reverseWords(self, s: str) -> str:
        tmp=''
        n=len(s)
        ans=[]
        for i,ch in  enumerate(s):
            if ch!=' ':tmp+=ch
            
            if i==n-1 or ch==' ':
                
                ans.append(tmp[::-1])
                tmp=''
            
        return ' '.join(ans)
```
