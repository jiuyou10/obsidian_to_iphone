# 541. 反转字符串 II

🔗 [题目链接](https://leetcode.cn/problems/reverse-string-ii/)   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:每2k步反转前k个字符

## 思路
#这样写更简洁
题目要求每2k个字符为一组，只反转每组的前k个字符。由于剩余字符的处理规则恰好与完整组的前k个字符反转一致（不足k个时全部反转，介于k和2k之间时只反转前k个），因此可以直接按2k步长遍历，每次反转i到i+k区间即可，无需分情况讨论。

## 关键点
- 步长设为2k，每次只处理当前区间的前k个字符
- 利用Python切片反转特性，直接原地修改列表

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
class Solution:
    def reverseStr(self, s: str, k: int) -> str:
        

        
        n=len(s)
        s=list(s)
        #其实不用分情况，可以综合
        for i in range(0,n,2*k):
            s[i:i+k]=s[i:i+k][::-1]
        return ''.join(s)
```
