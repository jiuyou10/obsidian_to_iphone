# 3775. 反转元音数相同的单词

## 我的代码

```python
class Solution:
    def reverseWords(self, s: str) -> str:
        arr=s.split()
        cnt=sum(1 for ch in arr[0] if ch in 'aeiou')
        for i in range(1,len(arr)):
            cur=sum(1 for ch in arr[i] if ch in 'aeiou')
            if cur==cnt:
                arr[i]=arr[i][::-1]
        
        return ' '.join(arr)
```
