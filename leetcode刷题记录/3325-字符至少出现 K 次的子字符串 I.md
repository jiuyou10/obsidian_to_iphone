# 3325. 字符至少出现 K 次的子字符串 I

🔗 [题目链接](https://leetcode.cn/problems/count-substrings-with-k-frequency-characters-i/)   📊 难度分 **1455**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+计数统计

## 思路
==主要记住这里可以用cnt[s[r]]来判断，不用max(cnt.values())==
题目要求统计所有包含至少一个字符出现至少k次的子串。通过滑动窗口维护一个右边界，当右边界字符出现次数达到k时，收缩左边界直到该字符次数小于k。此时，以当前右边界结尾的所有子串中，左边界之前的每个位置作为起点都能构成有效子串，因此答案累加左边界索引。

## 关键点
- 滑动窗口右边界每次扩展一个字符，只检查该字符是否达到k次
- 当右边界字符达到k次时，移动左边界直到该字符次数小于k，确保窗口内没有字符出现k次
- 每次右边界处理后，左边界索引l表示以当前右边界结尾的有效子串数量

## 复杂度
- 时间:O(n)
- 空间:O(1)（仅26个小写字母）

## 我的代码

```python
class Solution:
    def numberOfSubstrings(self, s: str, k: int) -> int:
        
        n=len(s)

        ans=l=0
        cnt=defaultdict(int)
        for r in range(n):
            cnt[s[r]]+=1
            #这里为什么可以用cnt[s[r]]来判断呢，因为当前只有其+1了
            
            while cnt[s[r]]>=k:
                cnt[s[l]]-=1
                l+=1
            #那如果不直接进入循环呢，这里的l-1其实可以是上一次循环之后退出时的l-1
            #比如152533，当到第二个5的时候，l移动到2，ans+=2，有1525，525
            #当到3的时候，不会进入循环，还是l=2,这里还是ans+=2，有15253和5253
            ans+=l
        return ans
```
