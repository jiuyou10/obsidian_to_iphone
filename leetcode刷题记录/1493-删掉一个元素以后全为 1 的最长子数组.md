# 1493. 删掉一个元素以后全为 1 的最长子数组

🔗 [题目链接](https://leetcode.cn/problems/longest-subarray-of-1s-after-deleting-one-element/)   📊 难度分 **1423**

> 💡 **关键词**:滑动窗口+最多一个0

## 思路
题目要求删除一个元素后得到最长的全1子数组，等价于在原数组中找一个最长的子数组，其中最多包含一个0（因为要删掉那个0）。使用滑动窗口维护窗口内0的个数不超过1，每次更新答案为窗口长度减1（删掉那个0），若窗口内没有0则减1对应必须删除一个元素的要求。

## 关键点
- 滑动窗口内0的个数cnt作为收缩条件，cnt>1时移动左指针
- 答案取r-l而不是r-l+1，因为必须删除一个元素（即使窗口全1也要减1）

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def longestSubarray(self, nums: List[int]) -> int:
        

        l=0
        n=len(nums)
        ans=0
        cnt=0
        for r in range(n):
            cnt+=nums[r]==0
            
            while cnt>1:
                cnt-=nums[l]==0
                
                l+=1
            #[l,r]要么全是1，要么有一个0，要删去一个，所以r-l+1-1
            ans=max(ans,r-l)
        
        return ans
```
