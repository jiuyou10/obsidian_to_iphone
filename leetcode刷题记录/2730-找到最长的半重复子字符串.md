# 2730. 找到最长的半重复子字符串

🔗 [题目链接](https://leetcode.cn/problems/find-the-longest-semi-repetitive-substring/)   📊 难度分 **1502**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+计数相邻相等对

## 思路
题目要求子字符串中至多有一对相邻相等字符，因此可以用滑动窗口维护当前窗口内相邻相等字符对的数量。当数量超过1时，移动左指针直到移除一对相邻相等字符，从而保证窗口始终合法。遍历过程中记录窗口的最大长度即可。

## 关键点
- 用变量`same`记录窗口内相邻相等字符对的数量，初始为0
- 当`same > 1`时，左指针右移，直到跳过最近的一对相邻相等字符，并将`same`重置为1

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def longestSemiRepetitiveSubstring(self, s: str) -> int:
        
        n=len(s)
        ans=1
        l=0
        same=0
        for i in range(1,n):
            same+=s[i]==s[i-1]
            if same>1:
                l+=1
                while s[l]!=s[l-1]:
                    l+=1
                #当找到相等的时候这里已经把l-1弹出去
                same=1
            
            ans=max(ans,i-l+1)
        return ans
```
