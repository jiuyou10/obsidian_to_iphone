# 1208. 尽可能使字符串相等

🔗 [题目链接](https://leetcode.cn/problems/get-equal-substrings-within-budget/)   📊 难度分 **1497**

> 💡 **关键词**:滑动窗口求最长子串

## 思路
题目本质是求满足总开销不超过预算的最长连续子串长度。将每个位置的开销计算出来，用滑动窗口维护当前窗口内的总开销，当总开销超过预算时移动左指针缩小窗口，每次更新最大长度即可。

## 关键点
- 将字符转换开销转化为整数数组，便于滑动窗口计算
- 滑动窗口右指针遍历，左指针在总开销超预算时右移

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def equalSubstring(self, s: str, t: str, maxCost: int) -> int:
        
        
        length=len(s)
        n=len(s)
        ans=0
        cur=0
        l=0
        for i in range(n):
            cur+=abs(ord(s[i])-ord(t[i]))
            while cur>maxCost:
                cur-=abs(ord(s[l])-ord(t[l]))
                l+=1
            
            ans=max(ans,i-l+1)
        return ans
```
