# 2799. 统计完全子数组的数目

🔗 [题目链接](https://leetcode.cn/problems/count-complete-subarrays-in-an-array/)   📊 难度分 **1398**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+计数统计

## 思路
题目要求子数组中不同元素数等于整个数组不同元素数。使用滑动窗口维护一个窗口，当窗口内不同元素数达到目标值时，收缩左边界并累加答案，因为此时左边界之前的每个位置作为起点都能与当前右边界构成完全子数组。

## 关键点
- 先计算整个数组的不同元素总数作为目标值
- 滑动窗口右移时更新计数，左移时删除计数为零的元素
- 每次满足条件时，左边界移动的次数即为以当前右边界结尾的完全子数组数量

## 复杂度
- 时间:O(n)
- 空间:O(k)，k为不同元素数

## 我的代码

```python
class Solution:
    def countCompleteSubarrays(self, nums: List[int]) -> int:
        
        hset=set(nums)
        setlen=len(hset)
        cnt=defaultdict(int)
        n=len(nums)
        ans=l=0
        for r in range(n):
            cnt[nums[r]]+=1

            while len(cnt)==setlen:
                cnt[nums[l]]-=1
                if cnt[nums[l]]==0:
                    del cnt[nums[l]]
                l+=1
        #其实不用tmp,直接用l，移动的几次，就加几,因为l只在满足条件时才右移收缩
        #右移的几次，左边界可以作为下标的元素就有几个，且可以用于下一位元素的判断
            ans+=l
        return ans
```
