# 917. 仅仅反转字母

🔗 [题目链接](https://leetcode.cn/problems/reverse-only-letters/)   📊 难度分 **1229**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:双指针原地反转字母

## 思路
题目要求只反转字母，非字母字符保持原位。使用双指针从两端向中间扫描，跳过非字母字符，遇到字母时交换，即可实现字母的原地反转，类似快速选择的分区思想。

## 关键点
- 双指针分别从左右两端向中间移动，确保只交换字母
- 跳过非字母字符时，需用 while 循环连续跳过
- 交换后同时移动双指针，避免重复处理

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def reverseOnlyLetters(self, s: str) -> str:
        
        s=list(s)
        

        #双指针两边扫描,类似快速选择
        l,r=0,len(s)-1
        while l<r:
            while l<r and not s[l].isalpha():
                l+=1
            while l<r and not s[r].isalpha():
                r-=1
            s[l],s[r]=s[r],s[l]
            l+=1
            r-=1
        

        return ''.join(s)
```
