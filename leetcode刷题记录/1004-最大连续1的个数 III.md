# 1004. 最大连续1的个数 III

🔗 [题目链接](https://leetcode.cn/problems/max-consecutive-ones-iii/)   📊 难度分 **1656**   ⏱️ 耗时 **7 分钟**

> 💡 **关键词**:滑动窗口 + 零计数控制

## 思路
题目本质是找最长的子数组，其中0的个数不超过k（因为可以翻转这些0为1）。使用滑动窗口，右指针扩展窗口，用计数器记录窗口内0的个数；当0的个数超过k时，左指针右移缩小窗口，直到0的个数满足条件。每次更新窗口长度最大值即可。

## 关键点
- 用计数器记录窗口内0的个数，而不是翻转操作本身
- 窗口内0的个数超过k时，移动左指针直到满足条件

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def longestOnes(self, nums: List[int], k: int) -> int:
        

        n=len(nums)
        cnt=ans=l=0
        for r in range(n):
            cnt+=nums[r]==0
            while cnt>k:
                cnt-=nums[l]==0
                l+=1
            ans=max(ans,r-l+1)
        return ans
```
