# 3090. 每个字符最多出现两次的最长子字符串

🔗 [题目链接](https://leetcode.cn/problems/maximum-length-substring-with-two-occurrences/)

> 💡 **关键词**:滑动窗口+计数数组

## 思路
题目要求每个字符最多出现两次，因此可以用滑动窗口维护一个合法区间。当右指针加入新字符导致某个字符出现次数超过2时，左指针右移直到该字符次数≤2，每次更新最大窗口长度。

## 关键点
- 用长度为26的数组记录窗口内每个字符的出现次数
- 当新字符导致计数>2时，移动左指针并减少对应字符计数，直到计数≤2

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def maximumLengthSubstring(self, s: str) -> int:
        cnt=[0]*26
        n=len(s)
        l=0
        ans=0
        for r in range(n):
            cur=ord(s[r])-ord('a')
            cnt[cur]+=1
            while cnt[cur]>2:
                cnt[ord(s[l])-ord('a')]-=1
                l+=1
            
            ans=max(ans,r-l+1)
        return ans
```
