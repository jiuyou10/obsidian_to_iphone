# 825. 适龄的朋友

🔗 [题目链接](https://leetcode.cn/problems/friends-of-appropriate-ages/)   📊 难度分 **1697**   ⏱️ 耗时 **30 分钟**

> 💡 **关键词**:双指针+年龄计数

## 思路
不好想，这道题
题目本质是统计满足条件的有序对(x,y)数量，条件可简化为：0.5*ages[x]+7 < ages[y] ≤ ages[x]。由于年龄范围仅1-120，可用计数数组统计每个年龄人数，再用双指针维护一个年龄区间，区间内所有年龄都满足向当前年龄r发送请求的条件，累加时注意减去自己发给自己的情况。

## 关键点
- 条件化简为 ages[y] 在 (0.5*ages[x]+7, ages[x]] 区间内
- 利用年龄范围小，用计数数组代替排序，双指针滑动窗口维护合法区间
- 计算请求数时，当前年龄r的人可以向区间内所有人发请求，但需减去自己（cnt[r]个）

## 复杂度
- 时间:O(n+120) ≈ O(n)
- 空间:O(120) ≈ O(1)

## 我的代码

```python
class Solution:
    def numFriendRequests(self, ages: List[int]) -> int:
        
        cnt=[0]*121
        ans=cur=l=0
        n=len(ages)
        for age  in ages:
            cnt[age]+=1
        for r in range(121):
            cur+=cnt[r]
            #遍历年龄
            #对于每一个r,找到满足条件的最小l,然后判断这个年龄区间有多少个人
            #而且每个age不一定只有一个人，可能有多个，所以需要相乘
            #age[l] 在区间 (0.5*r + 7, r]
            
            #这里不用while，直接if即可，因为r移动一位，l也最多移动一位,最多也就越界
一位，
            #还有这里也不需要判断l<r:也是因为当r移动一位时l才可能移动，且最多移动一
位，所以一定左端点小于右端点的
            if l<=0.5*r+7:
                cur-=cnt[l]
                l+=1
            
            if cur:#窗口里存在可以发送好友的用户
            #这个窗口也包含r在内，而r可以给同年龄的其他人发请求，但是不能给自己发
                ans+=cnt[r]*cur-cnt[r]
        return ans
```
