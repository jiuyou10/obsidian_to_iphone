# 61. 旋转链表

🔗 [题目链接](https://leetcode.cn/problems/rotate-list/)   ⏱️ 耗时 **11 分钟**

> 💡 **关键词**:成环后断开

## 思路
旋转链表本质上是将链表尾部的一段移动到头部。先遍历链表得到长度并找到尾节点，然后将链表成环，再根据 k 对长度取模找到新的头节点位置，最后断开环即可。

## 关键点
- 先将链表成环，避免处理复杂指针操作
- k 可能大于链表长度，需要取模
- 找到新的头节点后，前一个节点要置为 None

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
return None
        dummy=ListNode(next=head)
        prev=dummy
        tail=head
        n=1
        while tail.next:
            prev=tail
            tail=tail.next
            n+=1
        tail.next=head
        k=k%n
        tail=head
        prev=dummy
        cnt=n-k
        
        while cnt>0:
            prev=tail
            tail=tail.next
            cnt-=1
        print(tail,prev)
        newhead=tail
        prev.next=None
        return newhead
```
