# 2024. 考试的最大困扰度

🔗 [题目链接](https://leetcode.cn/problems/maximize-the-confusion-of-an-exam/)   ⏱️ 耗时 **9 分钟**

## 思路

题目要求最多改 k 个字符后能得到的最长全相同连续子串。最终要么全 T，要么全 F，分成两种策略分别用滑动窗口：

- **窗口1（→全 F）**：窗口内允许最多 k 个 T（把这些 T 改成 F），窗口长度就是可能的全 F 长度
- **窗口2（→全 T）**：窗口内允许最多 k 个 F（把这些 F 改成 T），窗口长度就是可能的全 T 长度

两个窗口同时滑动，取两种策略的最大值。

## 关键点
- 两个独立滑动窗口，分别跟踪 T 和 F 的"违规"数量
- `l1` 对应窗口1（允许 k 个 T），`l2` 对应窗口2（允许 k 个 F）
- 每次取 `max(i-l1+1, i-l2+1)` 更新答案

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 我的代码

```python
class Solution:
    def maxConsecutiveAnswers(self, answerKey: str, k: int) -> int:
        
        #如果只转换为T
        n=len(answerKey)
        ans=l1=l2=0
        x=y=0
        for i in range(n):
            cur=answerKey[i]
            x+=cur=='T'
            y+=cur=='F'
            while x>k:
                x-=answerKey[l1]=='T'
                l1+=1
            
            while y>k:
                y-=answerKey[l2]=='F'
                l2+=1
            
            ans=max(ans,i-l1+1,i-l2+1)
        
        return ans
```
