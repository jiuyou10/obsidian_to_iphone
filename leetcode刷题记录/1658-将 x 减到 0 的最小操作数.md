# 1658. 将 x 减到 0 的最小操作数

🔗 [题目链接](https://leetcode.cn/problems/minimum-operations-to-reduce-x-to-zero/)   📊 难度分 **1817**   ⏱️ 耗时 **39 分钟**

> 💡 **关键词**:滑动窗口找最长连续子数组

## 思路
这里没有想到怎么转化为滑动窗口，注意这种去掉两边的，可以反向转化
题目要求从两端移除元素使总和等于 x，等价于在数组中寻找一个最长的连续子数组，其和等于 total - x。因为移除的元素就是数组两端，剩下的中间部分就是连续子数组。使用滑动窗口遍历数组，当窗口和大于目标时收缩左边界，找到所有和等于目标的窗口，记录最大长度，最终用数组总长度减去该长度即为最小操作数。

## 关键点
- 将问题转化为求最长连续子数组和等于 total - x
- 滑动窗口维护窗口和，动态调整左右边界
- 注意处理 total < x 的边界情况

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def minOperations(self, nums: List[int], x: int) -> int:
        

        #反向，去掉两边，相当于找中间的连续数组和是否等于s-x
        total=sum(nums)
        if total<x:
            return -1
        l=0
        ans=-1
        n=len(nums)
        cur=0
        for i in range(n):
            cur+=nums[i]
            print(cur)
            while cur>total-x:
                cur-=nums[l]
                l+=1
            if cur==total-x:
                print(l,i,cur)
                ans=max(ans,i-l+1)
            
        
        if ans==-1:
            return -1
        return n-ans
```
