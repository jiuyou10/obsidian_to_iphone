# 3795. 不同元素和至少为 K 的最短子数组长度

🔗 [题目链接](https://leetcode.cn/problems/minimum-subarray-length-with-distinct-sum-at-least-k/)   📊 难度分 **1505**   ⏱️ 耗时 **95 分钟**

> 💡 **关键词**:滑动窗口+哈希表维护不同元素和

## 思路
初始看错题了，这里不能直接判断重复就去除，而是应该重复元素，进去时不计入总和，同理出来时也不计入总和
题目要求最短子数组，且子数组中不同元素的和至少为k，这提示使用滑动窗口。通过哈希表记录窗口内每个元素的出现次数，并维护当前窗口内不同元素的和，当和满足条件时尝试收缩左边界以寻找更短长度。

## 关键点
- 用哈希表记录元素出现次数，当某个元素首次出现时将其值加入和，当出现次数减为0时从和中减去
- 滑动窗口右边界扩展时更新哈希表和和，左边界收缩时检查元素是否变为0以更新和

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
class Solution:
    def minLength(self, nums: List[int], k: int) -> int:
        
        #最小长度不是1就是2，或者不存在
        #也可以单个元素的数组，最小长度是1
        if k<=max(nums):
            return 1
        ans=inf
        n=len(nums)
        cnt=defaultdict(int)
        s=l=0
        for i in range(n):
            cur=nums[i]
            cnt[cur]+=1
            if cnt[cur]==1:s+=cur
            while s>=k:
                if cnt[nums[l]]==1: s-=nums[l]
                cnt[nums[l]]-=1
                print(l,i)
                ans=min(ans,i-l+1)
                l+=1
        if ans==inf: return -1
        return ans
```
