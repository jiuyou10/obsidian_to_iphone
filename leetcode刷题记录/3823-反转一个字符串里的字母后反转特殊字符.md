# 3823. 反转一个字符串里的字母后反转特殊字符

🔗 [题目链接](https://leetcode.cn/problems/reverse-letters-then-special-characters-in-a-string/)   📊 难度分 **1250**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:双指针分别反转字母和特殊字符

## 思路
另一种解法，更简洁，比我自己写的
题目要求分别反转字母和特殊字符，但保持各自的位置不变。因此先遍历字符串，将字母和特殊字符的下标分别存入两个数组，然后对每个数组内的下标对应的字符进行反转即可。

## 关键点
- 分别存储字母和特殊字符的下标，而不是字符本身
- 反转时直接交换原字符串中对应下标的字符

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
def reverseByType(self, s: str) -> str:
        
        strarr=[]
        charr=[]
        s=list(s)
        n=len(s)
        for i in range(n):
            if s[i].islower():
                strarr.append(i)
            else:
                charr.append(i)
        
        #只存下标，直接交换s中下标对应的值
        def swap(t):
            l,r=0,len(t)-1
            while l<=r:
                s[t[l]],s[t[r]]=s[t[r]],s[t[l]]
                l+=1
                r-=1
            
        swap(strarr)
        swap(charr)
        
        return ''.join(s)
```
