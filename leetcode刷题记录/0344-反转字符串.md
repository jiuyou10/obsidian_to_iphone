# 344. 反转字符串

🔗 [题目链接](https://leetcode.cn/problems/reverse-string/)   ⏱️ 耗时 **3 分钟**

> 💡 **关键词**:双指针原地交换

## 思路
题目要求原地反转数组，不能使用额外空间。用左右双指针从两端向中间遍历，每次交换两个指针指向的元素，直到指针相遇或交错，即可完成反转。

## 关键点
- 使用两个指针分别指向数组首尾，逐步向中间移动
- 每次交换左右指针的元素，实现对称位置互换
- 循环条件为左指针小于等于右指针，确保所有元素都被处理

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def reverseString(self, s: List[str]) -> None:
        """
        Do not return anything, modify s in-place instead.
        """
        

        l,r=0,len(s)-1
        while l<=r:
            s[l],s[r]=s[r],s[l]
            l+=1
            r-=1
        
        return s
```
