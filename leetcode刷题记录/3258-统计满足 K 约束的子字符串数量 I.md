# 3258. 统计满足 K 约束的子字符串数量 I

🔗 [题目链接](https://leetcode.cn/problems/count-substrings-that-satisfy-k-constraint-i/)   📊 难度分 **1258**   ⏱️ 耗时 **9 分钟**

> 💡 **关键词**:滑动窗口+双指针计数

## 思路

题目要求统计所有 0 和 1 **不同时超过 k** 的子串数量。用一个滑动窗口维护合法区间，窗口收缩条件是 0 **且** 1 都超标。

- **合法性条件**：`x ≤ k 或 y ≤ k`（至少一个字符没超标）
- **收缩条件**：`x > k 且 y > k`（两个都超了才缩左边界）
- **计数方式**：每个右端点 i，以 i 结尾的合法子串数 = `i - l + 1`（窗口长度），累加即可

## 关键点
- 收缩条件是 **`and`** 不是 **`or`**：必须 0 和 1 **同时**超标才收缩；只要有一个没超 k，窗口就是合法的
- 每次右移后累加的是「以 i 结尾的所有合法子串」，不是窗口长度本身
- `x` 和 `y` 跟踪窗口内 0 和 1 的个数，和 [[2024-考试的最大困扰度]] 类似但收缩逻辑不同

## 复杂度
- 时间: O(n)
- 空间: O(1)

## 我的代码

```python
class Solution:
    def countKConstraintSubstrings(self, s: str, k: int) -> int:
        n=len(s)
        ans=0

        l=0
        ans=x=y=0
        for i in range(n):
            x+=s[i]=='0'
            y+=s[i]=='1'
            while x>k and y>k:
                x-=s[l]=='0'
                y-=s[l]=='1'
                l+=1
            ans+=i-l+1
        return ans
```
