# 930. 和相同的二元子数组

🔗 [题目链接](https://leetcode.cn/problems/binary-subarrays-with-sum/)   📊 难度分 **1592**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+前缀和转换

## 思路
重点标记，新思路
题目要求恰好等于 goal 的子数组个数，直接统计较复杂。可以转换为求“和大于等于 k 的子数组个数”减去“和大于等于 k+1 的子数组个数”，这样只需实现一个滑动窗口函数，统计每个右端点下满足条件的左端点数量即可。

## 关键点
- 将“恰好等于”转化为两个“大于等于”的差值，简化问题
- 滑动窗口维护窗口内元素和，当和大于等于 k 时收缩左边界，保证窗口内和严格小于 k
- 每个右端点对应的合法左端点数量就是当前左边界索引 l，累加即可

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def numSubarraysWithSum(self, nums: List[int], goal: int) -> int:
        #恰好等于k。转换为>=的个数减去>k的个数
        #也就是大于等于k的个数减去大于等于k+1的个数
         
        n=len(nums)

        def solve(k):
            ans=l=s=0
            for r in range(n):
                s+=nums[r]
                while l<=r and s>=k:
                    s-=nums[l]
                    l+=1
                ans+=l
            return ans
        
        return solve(goal)-solve(goal+1)
```
