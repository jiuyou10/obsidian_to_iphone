# 48. 旋转图像

🔗 [题目链接](https://leetcode.cn/problems/rotate-image/)

> 💡 **关键词**:分层旋转+四元素交换

## 思路
将矩阵看作由外到内的多个同心层，每次处理一层。对于每一层，将四个边上的对应元素进行顺时针旋转交换，即每个元素移动到下一个边的对应位置。通过循环处理所有层，实现整体旋转。

## 关键点
- 使用四个指针（l, r, t, b）标记当前层的边界
- 每层中，通过一个循环遍历该层边上的元素，每次交换四个对应位置的元素
- 每处理完一层，收缩边界，直到所有层处理完毕

## 复杂度
- 时间:O(n²)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        m,n=len(matrix),len(matrix[0])
        l,r=0,n-1
        t,b=0,m-1
        while True:

            for i in range(0,r-l):
                tmp=matrix[t][l+i]
                matrix[t][l+i]=matrix[b-i][l]
                matrix[b-i][l]=matrix[b][r-i]
                matrix[b][r-i]=matrix[t+i][r]
                matrix[t+i][r]=tmp
                
            l+=1
            r-=1
            t+=1
            b-=1
            if l>r or t>b:
                break
        return matrix
```
