# 151. 反转字符串中的单词

🔗 [题目链接](https://leetcode.cn/problems/reverse-words-in-a-string/)   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:分割+反转+拼接

## 思路
比我写的简洁
利用字符串的 split() 方法可以自动处理多余空格（包括前导、尾随和单词间多个空格），将单词提取为列表。然后反转列表顺序，再用单个空格 join 即可得到结果。

## 关键点
- split() 默认按任意空白字符分割，且自动去除空字符串
- 反转列表用切片 [::-1] 简洁高效
- join 用单个空格连接，保证结果无多余空格

## 复杂度
- 时间:O(n)
- 空间:O(n)

## 我的代码

```python
class Solution:
    def reverseWords(self, s: str) -> str:
        
        arr=s.split() #直接按空格分割为list，并且省去前后空格
        return ' '.join(arr[::-1])
```
