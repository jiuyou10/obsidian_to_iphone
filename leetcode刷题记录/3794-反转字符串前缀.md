# 3794. 反转字符串前缀

🔗 [题目链接](https://leetcode.cn/problems/reverse-string-prefix/)   📊 难度分 **1230**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:字符串切片反转拼接

## 思路
==注意这道题的切片时自动切断的，不会越界==
题目要求只反转前k个字符，其余部分保持不变。利用Python的切片特性，可以轻松提取前k个字符并反转，再与剩余部分拼接即可。

## 关键点
- 使用`s[:k]`获取前k个字符
- 使用`[::-1]`实现字符串反转
- 使用`+`将反转后的前缀与剩余后缀拼接

## 复杂度
- 时间:O(n)，n为字符串长度
- 空间:O(n)，创建新字符串

## 我的代码

```python
class Solution:
    def reversePrefix(self, s: str, k: int) -> str:
        
       

        return s[:k][::-1]+s[k:]
```
