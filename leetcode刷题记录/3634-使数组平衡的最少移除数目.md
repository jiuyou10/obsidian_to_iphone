# 3634. 使数组平衡的最少移除数目

🔗 [题目链接](https://leetcode.cn/problems/minimum-removals-to-balance-array/)   📊 难度分 **1453**

> 💡 **关键词**:排序+滑动窗口找最长平衡子数组

## 思路
题目要求移除最少元素使剩余数组平衡，等价于找到最长的平衡子数组。排序后，使用滑动窗口维护一个区间，保证区间内最大值（右端点）不超过最小值（左端点）的k倍，窗口长度即为当前最长平衡子数组长度。最终移除数等于总长度减最长平衡子数组长度。

## 关键点
- 排序后，平衡条件简化为 `nums[r] <= nums[l] * k`
- 滑动窗口右指针固定，左指针右移直到条件满足，保证窗口内所有元素都平衡

## 复杂度
- 时间:O(n log n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def minRemoval(self, nums: List[int], k: int) -> int:
        
        nums.sort()
        n=len(nums)
        l=0
        ans=0
        for r in range(n):
            cur=nums[r]
            while cur>nums[l]* k:
                l+=1
            
            ans=max(ans,r-l+1)
        
        return n-ans
```
