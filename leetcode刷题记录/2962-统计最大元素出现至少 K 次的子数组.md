# 2962. 统计最大元素出现至少 K 次的子数组

🔗 [题目链接](https://leetcode.cn/problems/count-subarrays-where-max-element-appears-at-least-k-times/)   📊 难度分 **1701**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+计数统计

## 思路
==题目中的“最大元素”指的是整个数组的最大值，而非子数组的最大值。==
因此只需统计包含该最大值至少k次的子数组数量。使用滑动窗口，右指针扩展窗口，当窗口内最大值出现次数达到k时，移动左指针缩小窗口，此时所有以当前右指针结尾、左边界小于等于l-1的子数组都满足条件，累加左指针位置即可。

## 关键点
- 明确“最大元素”是全局最大值，避免误解
- 滑动窗口维护计数，当计数达到k时收缩左边界
- 每次满足条件时，左指针位置l即为以当前右指针结尾的合法子数组数量

## 复杂度
- 时间:O(n)
- 空间:O(1)（只记录最大值计数，实际常数空间）

## 我的代码

```python
class Solution:
    def countSubarrays(self, nums: List[int], k: int) -> int:
        
        #无语，这里的最大值不是子数组的最大值，是nums数组的最大值
        
        n=len(nums)
        ans=l=0
        cnt=defaultdict(int)
        maxnum=max(nums)
        for r in range(n):
            cnt[nums[r]]+=1
            while cnt[maxnum]>=k:
                cnt[nums[l]]-=1
                l+=1
            #l-1作为满足条件的最后一个左边界
            ans+=l
        return ans
```
