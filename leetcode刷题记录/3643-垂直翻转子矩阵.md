# 3643. 垂直翻转子矩阵

🔗 [题目链接](https://leetcode.cn/problems/flip-square-submatrix-vertically/)   📊 难度分 **1235**   ⏱️ 耗时 **4 分钟**

> 💡 **关键词**:双指针逐行交换

## 思路
题目要求垂直翻转正方形子矩阵的行顺序，即第一行与最后一行交换，第二行与倒数第二行交换，以此类推。通过双指针从上下两端向中间移动，每次交换两行对应列范围内的所有元素，即可完成翻转。

## 关键点
- 确定子矩阵的边界：上边界为 x，下边界为 x+k-1，左边界为 y，右边界为 y+k-1
- 使用双指针 top 和 down 分别指向待交换的上下行，循环直到 top > down
- 利用 Python 的列表切片交换两行元素，简洁高效

## 复杂度
- 时间:O(k^2)，需要交换 k/2 行，每行交换 k 个元素
- 空间:O(1)，原地修改矩阵

## 我的代码

```python
class Solution:
    def reverseSubmatrix(self, grid: List[List[int]], x: int, y: int, k: int) 
-> List[List[int]]:
        
        top=x
        down=x+k-1
        l=y
        r=y+k-1
        while top<=down:
            grid[top][l:r+1],grid[down][l:r+1]=grid[down][l:r+1],grid[top][l:r
+1]
            top+=1
            down-=1
        
        return grid
```
