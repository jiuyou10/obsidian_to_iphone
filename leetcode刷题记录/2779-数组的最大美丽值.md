# 2779. 数组的最大美丽值

🔗 [题目链接](https://leetcode.cn/problems/maximum-beauty-of-an-array-after-applying-operation/)   📊 难度分 **1638**   ⏱️ 耗时 **30 分钟**

> 💡 **关键词**:排序+滑动窗口找最大重叠区间

## 思路
每个元素可以变成 [nums[i]-k, nums[i]+k] 内的任意值，所以问题转化为：找到最大的下标集合，使得这些区间有公共交集。排序后，用滑动窗口维护一个区间，保证窗口内所有区间都重叠（即最左区间的右端点 >= 最右区间的左端点），窗口大小即为可能的美丽值。

## 关键点
- 排序后，区间重叠条件简化为：nums[l] + k >= nums[i] - k
- 滑动窗口左指针 l 单调右移，保证 O(n) 时间复杂度

## 复杂度
- 时间:O(n log n)（排序主导）
- 空间:O(log n)（排序栈空间）

## 我的代码

```python
class Solution:
    def maximumBeauty(self, nums: List[int], k: int) -> int:
        
        nums.sort()
        n=len(nums)
        ans=l=0
        for i in range(n):
            while nums[i]-k>nums[l]+k:
                l+=1
            ans=max(ans,i-l+1)
        return ans
```
