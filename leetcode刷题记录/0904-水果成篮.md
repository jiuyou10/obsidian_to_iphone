# 904. 水果成篮

🔗 [题目链接](https://leetcode.cn/problems/fruit-into-baskets/)   📊 难度分 **1516**   ⏱️ 耗时 **6 分钟**

> 💡 **关键词**:滑动窗口+哈希表计数

## 思路
题目本质是求最长连续子数组，其中最多包含两种不同元素。使用滑动窗口维护一个窗口，窗口内水果种类不超过2种，通过哈希表记录每种水果的个数。当窗口内种类超过2时，移动左指针缩小窗口直到种类数恢复合法，每次更新最大窗口长度。

## 关键点
- 滑动窗口右指针遍历数组，左指针动态收缩
- 哈希表记录窗口内每种水果的计数，用于判断种类数是否超过2
- 左指针移动时需更新计数，当计数为0时从哈希表中删除该种类

## 复杂度
- 时间:O(n)，每个元素最多被左右指针各访问一次
- 空间:O(1)，哈希表最多存储3种水果的计数

## 我的代码

```python
class Solution:
    def totalFruit(self, fruits: List[int]) -> int:
        
        n=len(fruits)
        ans=tmp=l=0
        cnt=defaultdict(int)
        for i in range(n):
            cnt[fruits[i]]+=1
            tmp+=1
            while len(cnt)>2:
                cnt[fruits[l]]-=1
                tmp-=1
                if cnt[fruits[l]]==0:
                    del cnt[fruits[l]]
                l+=1
            ans=max(ans,tmp)
        
        return ans
```
