# 1861. 旋转盒子

🔗 [题目链接](https://leetcode.cn/problems/rotating-the-box/)   📊 难度分 **1537**   ⏱️ 耗时 **145 分钟**

> 💡 **关键词**:逐行统计石头+旋转填充

## 思路
旋转后每列对应原行，重力使石头沉到障碍物或底部。逐行遍历原矩阵，统计连续石头数量，遇到障碍物或行尾时，从当前位置倒序填充石头，模拟重力下落。同时将旋转后的字符直接填入结果矩阵对应位置。

## 关键点
- 原地统计石头数量，遇到障碍物或行尾再批量填充，避免逐个模拟下落
- 旋转后坐标映射：原(i,j) -> 新(j, m-1-i)
- 障碍物和空位在遍历时直接填入结果，石头先清空再统一填充

## 复杂度
- 时间:O(m*n)
- 空间:O(m*n)

## 我的代码

```python
class Solution:
    def rotateTheBox(self, boxGrid: List[List[str]]) -> List[List[str]]:
        
        #记录每一行，障碍物之间的石头数量，遇到一个障碍物就倒着放石头
        m,n=len(boxGrid),len(boxGrid[0])
        ans=[['']*m for i in range(n)]
        for i,row in enumerate(boxGrid):
            cnt=0
            
            for j,ch in enumerate(row):
                #遇到石头计数+1并且变为空
                if ch=='#':
                    cnt+=1
                    ch='.'
                ans[j][-i-1]=ch
                #如果到底了或者下一个是障碍物
                if j==n-1 or row[j+1]=='*':
                    for k in range(j,j-cnt,-1):
                        #i行k列转换为k行倒数i列
                        ans[k][-i-1]='#'
                        cnt=0
        return ans
```
