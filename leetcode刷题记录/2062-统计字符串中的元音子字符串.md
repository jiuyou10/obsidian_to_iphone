# 2062. 统计字符串中的元音子字符串

🔗 [题目链接](https://leetcode.cn/problems/count-vowel-substrings-of-a-string/)   📊 难度分 **1458**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+哈希表计数

## 思路
题目要求子串仅由元音组成且包含全部五种元音，因此可以用滑动窗口维护当前窗口内的元音种类。当窗口内恰好包含五种元音时，不断收缩左边界，每收缩一次就多一个有效子串，并将这个计数累加到答案中。遇到辅音时重置窗口和计数。

## 关键点
- 用哈希表记录窗口内各元音的出现次数，当哈希表大小为5时表示包含全部五种元音
- 每次满足条件时，通过移动左边界并累加tmp来统计以当前右边界结尾的所有有效子串数量

## 复杂度
- 时间:O(n)，其中n为字符串长度，每个字符最多被访问两次
- 空间:O(1)，哈希表最多存储5个元音字符

## 我的代码

```python
class Solution:
    def countVowelSubstrings(self, word: str) -> int:

        n=len(word)
        ans=l=0
        cnt=defaultdict(int)
        tmp=0
        for r in range(n):
            if word[r] not in ('a','e','i','o','u'): 
                l=r+1
                cnt.clear()
                tmp=0
                continue
            cnt[word[r]]+=1
            #当满足条件时推进l，每推进一次有效左边界就+1,且可以被下一位判断时继承
            while len(cnt)==5 :
                cnt[word[l]]-=1
                if cnt[word[l]]==0:
                    del cnt[word[l]]
                l+=1
                tmp+=1
            ans+=tmp
        return ans
```
