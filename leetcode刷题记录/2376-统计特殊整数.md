# 2376. 统计特殊整数

🔗 [题目链接](https://leetcode.cn/problems/count-special-integers/)   📊 难度分 **2120**

> 💡 **关键词**:数位DP+状态压缩

## 思路
题目要求统计[1,n]中每位数字互不相同的数，这类“按位统计、有上界限制”的计数问题通常用数位DP解决。核心是逐位构造数字，用位掩码记录已使用的数字，同时通过islimit控制当前位是否受n的对应位限制，通过isstart标记是否已开始填非零数字（避免前导零影响计数）。

## 关键点
- 用二进制掩码mask记录0-9哪些数字已被使用，判断重复
- islimit控制上界，isstart处理前导零，保证正确统计所有长度小于n位数的数
- 记忆化搜索缓存(i,mask,islimit,isstart)状态，避免重复计算

## 复杂度
- 时间:O(len(n) * 2^10 * 2 * 2) ≈ O(len(n) * 1024)
- 空间:O(len(n) * 2^10)

## 我的代码

```python
class Solution:
    def countSpecialNumbers(self, n: int) -> int:
        #高位相同，继续往低位找
        
        #用位编码来表示是否某一位已经存在
        s=str(n)
        slen=len(s)
        @cache
        def dfs(i,mask,islimit,isstart):
            if i==slen:
                return 1 if isstart else 0
                #要保证i前面填入非0的数字了，若全是0，则不可以
            res=0
            if not isstart: #跳过当前i
                #前面都是0，当前也可以填入0，继续往后走
                #跳过当前位，则后面islimit就是false,isstart也是false
                res=dfs(i+1,mask,False,False)
            low=0 if isstart else 1
            up=int(s[i]) if islimit else 9
            for d in range(low,up+1):
                if mask>>d&1==0:
                    #d不重复
                    res+=dfs(i+1,mask|(1<<d),islimit and d==up,True)
            return res

        return dfs(0,0,True,False)
```
