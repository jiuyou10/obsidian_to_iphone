# 2958. 最多 K 个重复元素的最长子数组

```python
class Solution:
    def maxSubarrayLength(self, nums: List[int], k: int) -> int:
        
        n=len(nums)
        ans=l=0
        cnt=defaultdict(int)
        
        for i in range(n):
            cur=nums[i]
            cnt[cur]+=1

            while cnt[cur]>k:
                left=nums[l]
                cnt[left]-=1
                l+=1

            ans=max(ans,i-l+1)
        return ans
```
