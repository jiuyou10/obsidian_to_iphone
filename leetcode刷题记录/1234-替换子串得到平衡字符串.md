# 1234. 替换子串得到平衡字符串

🔗 [题目链接](https://leetcode.cn/problems/replace-the-substring-for-balanced-string/)   📊 难度分 **1878**   ⏱️ 耗时 **10 分钟**

> 💡 **关键词**:滑动窗口+计数平衡

## 思路
题目要求通过替换一个子串使整个字符串平衡，等价于在子串外的剩余部分中，每个字符出现次数不超过 n/4。因此用滑动窗口遍历所有子串，维护窗口外字符计数，当窗口外所有字符计数 ≤ n/4 时，当前窗口即为一个可行替换，更新最小长度。

## 关键点
- 滑动窗口内是待替换子串，窗口外是剩余部分，维护窗口外字符计数
- 窗口外字符计数 ≤ n/4 是窗口可行的条件，此时窗口内可通过替换补足缺失字符

## 复杂度
- 时间:O(n)
- 空间:O(1)

## 我的代码

```python
class Solution:
    def balancedString(self, s: str) -> int:
        n=len(s)
        avg=n//4
        cnt=Counter(s)
        
        if len(cnt)==4 and max(cnt.values())==avg:
            return 0
        ans=n
        l=0
        for r in range(n):
            cnt[s[r]]-=1
            #cnt记录的是去除子串剩下的部分的个数,要保证剩下部分每个元素的个数都小于等
于avg

            while max(cnt.values())<=avg:
                ans=min(ans,r-l+1)
                cnt[s[l]]+=1
                l+=1
        
        return ans
```
